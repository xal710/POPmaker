# POP用フォント

## 金額（Pop-Price.ttf）

POP画像の**金額部分**に使うフォントです。

### 差し替え方法

1. 入手したフォントファイル（`.otf` / `.ttf` / `.woff` / `.woff2`）を **`Pop-Price.ttf`** という名前でこのフォルダに上書き保存する  
   - 拡張子が `.otf` でも、ファイル名は **`Pop-Price.ttf` のまま** で問題ありません（中身が OpenType 形式なら読み込めます）
   - 別名で置きたい場合は `src/utils/popFonts.ts` の `POP_PRICE_FONT.url` を変更してください
2. サイトをリロード（本番の場合はデプロイ後に **Ctrl+Shift+R**）
